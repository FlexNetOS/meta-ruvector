# Ruvector Tiny Dancer Node

[![Crates.io](https://img.shields.io/crates/v/ruvector-tiny-dancer-node.svg)](https://crates.io/crates/ruvector-tiny-dancer-node)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)

**Node.js bindings for Tiny Dancer neural routing via NAPI-RS.**

`ruvector-tiny-dancer-node` provides native Node.js bindings for embedding-based AI agent/model routing. It runs FastGRNN neural inference at native speed to score candidates and decide whether a cheap ("lightweight") model is good enough. Part of the [Ruvector](https://github.com/ruvnet/ruvector) ecosystem.

## Why Tiny Dancer Node?

- **Native Performance**: Rust speed in Node.js
- **Async/Await**: `route()` runs on a worker thread (non-blocking)
- **TypeScript**: Type definitions generated by NAPI-RS
- **Hot Reload**: Reload the model from disk without restarting (`reloadModel()`)
- **Circuit Breaker**: Built-in health gating (`circuitBreakerStatus()`)

## What It Does

Tiny Dancer is an **embedding router**. You give it a query embedding and a list of
candidates (each with its own embedding), and it returns routing decisions: which
candidate, with what confidence, and whether to use the lightweight model.

> **Honesty note:** this binding does **not** take raw text. There is no built-in
> tokenizer, sentiment analysis, or feature-extraction-from-strings. You bring your
> own embeddings. The router scores embeddings; it does not embed text for you.

## Quick Start

### Basic Routing

```typescript
import { Router } from '@ruvector/tiny-dancer-node';

// Create a router (the only required field is modelPath)
const router = new Router({
  modelPath: './models/fastgrnn.safetensors',
  confidenceThreshold: 0.85,
  maxUncertainty: 0.15,
  enableCircuitBreaker: true,
});

// Route a request: query embedding + candidate embeddings
const response = await router.route({
  queryEmbedding: new Float32Array([0.1, 0.2 /* ... */]),
  candidates: [
    { id: '1', embedding: new Float32Array([/* ... */]) },
    { id: '2', embedding: new Float32Array([/* ... */]) },
  ],
});

const top = response.decisions[0];
console.log(`Route to candidate ${top.candidateId} (confidence: ${top.confidence})`);
console.log(`Use lightweight model: ${top.useLightweight}`);
console.log(`Inference time: ${response.inferenceTimeUs} µs`);
```

### Hot-Reloading the Model

```typescript
import { Router } from '@ruvector/tiny-dancer-node';

const router = new Router({ modelPath: './models/fastgrnn.safetensors' });

// After replacing the file on disk, reload it without restarting:
await router.reloadModel();
```

### Circuit Breaker Status

```typescript
import { Router } from '@ruvector/tiny-dancer-node';

const router = new Router({ modelPath: './models/fastgrnn.safetensors' });

// true = circuit closed (healthy); false = open (unhealthy); null = disabled
const isHealthy = router.circuitBreakerStatus();
```

### Training a Router (DRACO dataset)

`trainRouter()` trains a FastGRNN from a DRACO dataset (query embedding +
per-model quality scores) and writes a `.safetensors` file consumable by `new Router({ modelPath })`.

```typescript
import { trainRouter, Router } from '@ruvector/tiny-dancer-node';

const rows = [
  { embedding: [0.1, 0.2, /* ... */], scores: { haiku: 0.9, opus: 0.95 } },
  // ... more rows
];

const result = await trainRouter(rows, { haiku: 1, opus: 15 }, {
  outputPath: './router.safetensors',
  inputDim: 8,   // must equal the embedding length
  epochs: 40,
});

console.log(`Epochs run: ${result.epochsRun}`);
console.log(`Train accuracy: ${result.trainAccuracy}`);
console.log(`Validation accuracy: ${result.valAccuracy}`);
console.log(`Model written to: ${result.modelPath} (${result.modelBytes} bytes)`);

const router = new Router({ modelPath: result.modelPath });
```

### Raw Scoring

`score()` loads a trained model and runs a raw forward pass on an embedding
(matching what `trainRouter` was trained on). The returned value is a sigmoid in
`0..1`: high means "the cheap model is good enough"; low means route to a stronger model.

```typescript
import { score } from '@ruvector/tiny-dancer-node';

const s = await score('./router.safetensors', [0.1, 0.2, /* ... */]);
console.log(`Cheap-model-good-enough score: ${s}`);
```

## API Reference

### `Router`

```typescript
class Router {
  constructor(config: RouterConfig);
  route(request: RoutingRequest): Promise<RoutingResponse>;
  reloadModel(): Promise<void>;
  circuitBreakerStatus(): boolean | null;
}
```

### Free Functions

```typescript
function trainRouter(
  rows: DracoRowJs[],
  prices: Record<string, number>,
  options: TrainRouterOptions,
): Promise<TrainRouterResult>;

function score(modelPath: string, embedding: number[]): Promise<number>;

function version(): string;
function hello(): string;
```

### Types

```typescript
interface RouterConfig {
  modelPath: string;
  confidenceThreshold?: number;     // default 0.85
  maxUncertainty?: number;          // default 0.15
  enableCircuitBreaker?: boolean;   // default true
  circuitBreakerThreshold?: number; // default 5
  enableQuantization?: boolean;     // default true
  databasePath?: string;
}

interface Candidate {
  id: string;
  embedding: Float32Array;
  metadata?: string;        // JSON string
  createdAt?: number;
  accessCount?: number;
  successRate?: number;
}

interface RoutingRequest {
  queryEmbedding: Float32Array;
  candidates: Candidate[];
  metadata?: string;        // JSON string
}

interface RoutingDecision {
  candidateId: string;
  confidence: number;
  useLightweight: boolean;
  uncertainty: number;
}

interface RoutingResponse {
  decisions: RoutingDecision[];
  inferenceTimeUs: number;
  candidatesProcessed: number;
  featureTimeUs: number;
}

interface DracoRowJs {
  embedding: number[];
  scores: Record<string, number>;
}

interface TrainRouterOptions {
  outputPath: string;
  inputDim: number;       // must equal the embedding length
  hiddenDim?: number;     // default 12
  epochs?: number;        // default 40
  learningRate?: number;  // default 0.05
  tolerance?: number;     // DRACO label tolerance, default 0.05
}

interface TrainRouterResult {
  epochsRun: number;
  trainLoss: number;
  trainAccuracy: number;
  valAccuracy: number;
  modelPath: string;
  modelBytes: number;
}
```
